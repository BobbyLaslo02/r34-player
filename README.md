R34 Player is a desktop app that turns browsing Rule34 into a Netflix/YouTube-style experience. Media loads in a scrollable grid feed with a built-in video player, playlist, and library system — all wrapped in a dark theme with privacy protections.

## Features

- Browse feed with Latest / Popular / Longest sorts
- YouTube-style player with playlist, prev/next, auto-advance
- Video-only filter (fetches 1000 posts, filters client-side)
- Library system to save and shuffle your favorite posts
- Multi-tag search with autocomplete and recent tags
- Simple search (joins words with underscore for single-tag queries)
- Genre cards with thumbnails and favorites
- File size displayed on every card and in the player
- Fullscreen mode for video/image container
- Volume slider and mute toggle
- VPN gate — blocks all content unless a VPN tunnel is active
- Proceed-anyway bypass with confirmation
- User-selectable color themes (5 themes)
- Auto-updater with GitHub Releases
- GoFile cloud upload — save media directly to your GoFile account
- Cross-platform: Windows (NSIS + Portable) and Linux (AppImage + Pacman)
- All data stored locally (localStorage) — nothing uploaded without your action
